Color Palette

Six tokens form the entire visual language. Primary pink drives warmth and brand recognition; secondary mint provides breathing room and contrast; CTA yellow creates urgency; nav blue anchors trust; contrast purple is used sparingly for micro-accents only.

Usage Rules

✅ Do
Use #FFD532 only for primary CTA buttons
Pair pink bg with #fff or #3f3f3f text
Use mint/secondary for footer and eyebrow tags
Keep #8224E3 to ≤ 5% of any page view
❌ Don't
Use CTA yellow as a background color for sections
Place white text on mint/secondary backgrounds
Use more than 3 colors in a single component
Use contrast purple for large text blocks

Pages · 03

Motion & Animation

Motion serves clarity, not decoration. All animations respect prefers-reduced-motion. Default easing is ease-out for entries, ease-in-out for hovers.

Page Load / Hero
Eyebrow tag fades in (opacity 0→1, 300ms). H1 slides up 20px + fades (500ms, delay 100ms). Body text fades (400ms, delay 300ms). Buttons fade (300ms, delay 500ms). Hero image slides in from right 40px (600ms, delay 200ms).
Scroll Reveal (IntersectionObserver)
Cards: translateY(24px) → 0, opacity 0→1, stagger 80ms per card. Trust badges: same. Threshold: 0.15. Once played, does not replay.
Button Hover
translateY(-2px) + box-shadow 0 8px 24px rgba(0,0,0,.15). Duration: 160ms ease-out. Active: translateY(0), 80ms. All buttons consistent.
Service Card Hover
translateY(-4px) + box-shadow 0 12px 32px rgba(0,0,0,.12). Duration: 200ms ease-out. Icon circle rotates 5deg. No color change.
Nav Scroll Effect
On scroll > 60px: navbar adds box-shadow 0 4px 20px rgba(0,0,0,.15) and slight blur backdrop. Logo card scales down from overflow to flush (subtle transform). Duration: 300ms.
Reduced Motion
@media (prefers-reduced-motion: reduce) — All transforms removed. Only opacity transitions preserved (max 200ms). No scroll animations.

Easing Reference

Entry animations: cubic-bezier(0.16, 1, 0.3, 1) — fast out, settles soft
Hover / interactive: ease-out — quick start, controlled end
Exit / dismiss: ease-in — accelerates to exit
Scroll animations: ease-out · 400ms max duration
